Biblioteka do wysyłki i zarządzania SMSami dla frameworku [Laravel](https://laravel.com/) przez

## Co w środku?
* Pełna implementacja REST API JustSend *(15.06.2020)*
* Prostota użycia
* Kanał powiadomień
* Komendy Artisana do zarządzania SMSami
* Logi z wysyłki
* Konfiguracja
* `OPCJONALNE` Zapis do bazy wysłanych wiadomości, wraz ze statusem powodzenia
* `OPCJONALNE` Panel do podglądu statystyk wysyłki
